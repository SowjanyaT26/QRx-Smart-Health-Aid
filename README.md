# QRx – Smart Health Aid

QRx – Smart Health Aid is a web-based emergency medical information system that enables quick access to essential health information through a unique QR code.

## Features

- User registration and secure login
- Emergency health profile management
- Unique emergency QR code generation
- User-controlled privacy settings
- QR activation and deactivation
- Automatic QR expiry
- Emergency QR access history
- Public emergency profile for QR scanning
- Password reset functionality

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Firebase Firestore
- Firebase Authentication
- qrcode.react
- Lucide React
- Vercel

## How It Works

1. The user creates an account and adds their emergency health information.
2. The system generates a unique QR-linked emergency profile.
3. The user controls which information is publicly visible.
4. The QR code can be scanned to access the permitted emergency information.
5. Users can manage QR availability, set an expiry time, and view access history.

## Purpose

QRx aims to provide quick access to essential medical information during emergency situations while giving users control over the privacy and availability of their data.
